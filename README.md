# MOKO SE - Установка зависимостей

Этот репозиторий содержит скрипт для установки всех необходимых Python-библиотек для корректной работы и выполнения скриптов на платформе автоматизации **MOKO SE**.

## Установка

Для установки всех необходимых зависимостей просто **запустите скачанный бат-файл** (файл с расширением `.bat`).

Скрипт автоматически установит и обновит все перечисленные ниже библиотеки.

## Описание библиотек

Ниже приведен список всех устанавливаемых пакетов и их краткое описание.

### Основные библиотеки

Это ключевые пакеты, которые предоставляют основной функционал для автоматизации и обработки данных.

*   **Requests**: Основная библиотека для отправки команд и запросов в MOKO SE по сети (HTTP).
*   **Numpy**: Используется для сложных математических вычислений, например, для формирования синусоидальных сигналов с разными частотами.
*   **Selenium**: Мощный инструмент для автоматизации действий в веб-браузере. Позволяет управлять браузером для тестирования или сбора данных с веб-сайтов.
*   **BeautifulSoup4**: Предназначена для извлечения (парсинга) данных из HTML и XML файлов. Отлично работает в связке с Selenium и Requests.
*   **PyQt5**: Один из самых популярных наборов инструментов для создания полноценных настольных приложений с графическим интерфейсом (GUI).
*   **Pandas**: Библиотека для обработки и анализа данных. Позволяет удобно работать с таблицами, временными рядами и другими структурированными данными.
*   **Openpyxl**: Позволяет читать и записывать файлы Excel (`.xlsx`).
*   **Pillow**: Библиотека для работы с изображениями: открытия, изменения, сохранения в различных форматах.
*   **python-telegram-bot**: Позволяет создавать ботов для мессенджера Telegram и управлять ими.
*   **lxml**: Очень быстрая библиотека для обработки XML и HTML, часто используется вместе с BeautifulSoup для ускорения парсинга.

### Вспомогательные библиотеки и зависимости

Эти пакеты, как правило, устанавливаются автоматически, так как от них зависит работа основных библиотек. Они обеспечивают базовую функциональность для работы с сетью, кодировками, временем и т.д.

*   **anyio, httpcore, httpx, h11, urllib3**: Набор современных библиотек для выполнения асинхронных и синхронных сетевых запросов.
*   **certifi**: Предоставляет актуальные SSL-сертификаты для безопасных HTTPS-соединений.
*   **charset-normalizer, idna**: Помогают корректно работать с разными кодировками и международными доменными именами в ссылках.
*   **et-xmlfile**: Вспомогательный пакет для работы с XML, используется библиотекой `openpyxl`.
*   **python-dateutil, pytz, tzdata**: Обеспечивают расширенную функциональность для работы с датами и часовыми поясами.
*   **six**: Обеспечивает совместимость кода между разными версиями Python.
*   **sniffio**: Помогает определять, какая асинхронная библиотека используется в проекте.
